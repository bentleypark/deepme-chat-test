# Deep_me 감정대화 테스트 페이지

Deep_me 백엔드 서버와 WebSocket으로 연결하여 감정대화를 테스트할 수 있는 웹 페이지입니다.

## 사용 방법

1. 웹 페이지 열기: https://deepme123.github.io/deepme-chat-test/
2. "연결" 버튼 클릭
3. 메시지 입력 후 Enter로 전송
4. 실시간 AI 응답 확인

## 기술 스택

- WebSocket (wss://deep-me-v1.onrender.com/ws/emotion)
- 바닐라 JavaScript
- GitHub Pages
